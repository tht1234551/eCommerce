# 인텔리제이 단축키

| 기능                   | 단축키                 |
|:---------------------|:--------------------|
| 코드 정리                | ctrl + alt + l      |
| 변수 자동완성              | ctrl + alt + v      |
| 파일 새로 만들기            | ctrl + alt + insert |
| 프로젝트 영역과 에디터 영역 이동하기 | ctrl + tap          |
| 안쓰는 import 제거        | ctrl + alt + o      |
